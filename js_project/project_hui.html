菜品介绍--回锅肉

准备时间：30分钟-1小时

制作时间：10-20分钟

用餐人数：2人

食材

  • 主料
  • 五花肉400g
  • 青蒜 250g
  • 辅料
  • 葱适量
  • 姜适量
  • 蒜适量
  • 干红辣椒适量
  • 花椒适量
  • 郫县豆瓣一勺
  • 料酒一勺
  • 糖适量
步骤
带皮五花肉冷水下锅加入葱段、姜片花椒7、8粒，黄酒适量煮开。
撇净浮沫，煮至八成熟，取出自然冷却。
下入肉片煸炒，至肉片颜色变透明，边缘略微卷起
适当的加入少许酱油或甜面酱调色，与肉片一起翻炒均匀
下入青蒜，点少许料酒，糖调好味道即可出锅
